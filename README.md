# ETE Lab & Assignment Cover Page Generator

Web application for generating cover pages for lab reports and assignments for the Electronics & Telecommunication Engineering department at RUET.

## Features
- Live preview
- Teacher selection dropdown
- Auto-save student information
- A4 PDF format
- Support for both Lab Reports and Assignments

## Live Demo
[View Live App](https://mahbuburrahmantabib.github.io/ete-cover-generator/)

## Usage
1. Fill in student information
2. Select teacher from dropdown
3. Enter course and experiment/assignment details
4. Preview updates in real-time
5. Download as PDF
